# Accessibility Notes – User Story 2

## What I checked
- Keyboard navigation on homepage
- Keyboard navigation on category selection
- FAQ search field usability
- FAQ accordion interaction
- Visible focus states
- Skip link behavior
- Admin dashboard instructions for drag-and-drop

## Improvements made
- Replaced non-semantic clickable card behavior with more accessible clickable controls
- Improved homepage student selection cards
- Improved category cards for keyboard and screen reader support
- Added clearer admin drag-and-drop instructions
- Kept visible focus indicators and skip link support

## Remaining limitations
- Drag-and-drop in the admin dashboard is still mainly mouse-based
- Full multilingual translation support is not implemented yet
- Full screen reader testing should still be done before final launch

## Manual test steps
1. Use Tab and Shift+Tab to move through the homepage
2. Open Current Student and Future Student pages using keyboard only
3. Search for FAQ terms using keyboard
4. Expand and collapse accordions using keyboard
5. Test admin page instructions and button access
