# Accessibility and Translation Notes

## User Story 2

As a Running Start student, I want the portal to be easier to navigate and accessible, so that I can use it successfully across devices and support tools.

## Features Added

- Keyboard-accessible accessibility toolbar
- Text size controls: A- and A+
- High contrast display mode
- Readable font mode
- Read page aloud support using browser speech synthesis
- Google Translate language selector
- Reset settings option

## Testing Completed

- Tabbed through the accessibility toolbar using keyboard only
- Tested text size increase and decrease
- Tested high contrast mode
- Tested readable font mode
- Tested reset settings
- Tested read page and stop reading
- Confirmed the translation dropdown appears when Google Translate loads

## Notes

Google Translate may not always load on localhost. It is expected to work more reliably after deployment on Vercel.
