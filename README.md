### Team Name: Why Are You Running?

**Client:** Running Start Department – Green River College  
**Capstone Duration:** Jan–June 2026

### Project Description
A web-based FAQ and digital resource support tool designed to help Running Start students navigate the program, access answers quickly, and reduce repeated questions to Running Start staff.

### Features (Planned)
- Student-friendly FAQ browsing
- Organized categories (Enrollment, Advising, Credits, Deadlines)
- Mobile responsive design
- Editable content for staff

### Future Stretch Goals
- Analytics dashboard
- Chatbot integration
- Auto-email responses
